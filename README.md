🌸 Blossom Flower Shop Website

A dynamic, full-stack e-commerce web application built using PHP and MySQL.

Blossom Flower Shop allows customers to browse various floral arrangements, manage a shopping cart, and place orders online.

The system also provides administrators with a backend dashboard to manage products, categories, inventory, and customer orders efficiently.

The project demonstrates responsive web design, database integration, user authentication, shopping cart functionality, and online order management for a real-world flower shop business.

🚀 Features

## Customer-Facing Features

### Product Catalog

Browse flowers by category such as Weddings, Birthdays, and Anniversaries with detailed descriptions, pricing, and product images.

### Shopping Cart

Add products to the cart, update item quantities, or remove items dynamically before checkout.

### Secure Checkout

User authentication system with Sign Up and Login functionality using secure password hashing.

### Order History

Registered users can track previous orders and monitor current order fulfillment status.
